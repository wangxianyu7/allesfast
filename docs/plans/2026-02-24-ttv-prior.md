**Goal:** Add a Gaussian chi2 penalty from observed transit midtimes (`*.{companion}.ttv` files) to constrain orbital period and epoch without entering full TTV-fitting mode.

**Architecture:** Scan `datadir` for `*.{companion}.ttv` files at BASEMENT init and store in `self.ttv_data`. Add `use_ttv_prior` boolean setting. Inside `calculate_external_priors`, compute the linear-ephemeris residual chi2 and accumulate into `lnp`.

**Tech Stack:** NumPy (file loading, round, sum), existing `config.BASEMENT` / `calculate_external_priors` pattern.

```python
        if _arr.ndim == 1:

        self.ttv_data[_companion] = (_arr[:, 0], _arr[:, 1])
```

```python
#::: TTV linear-ephemeris prior
if config.BASEMENT.settings.get('use_ttv_prior', False):
    for _comp, (_T_obs, _sigma) in config.BASEMENT.ttv_data.items():
        _tc = params.get(_comp + '_epoch')
        _P  = params.get(_comp + '_period')
        if _tc is None or _P is None or _P <= 0:
            return -np.inf
        _N      = np.round((_T_obs - _tc) / _P)
        _T_pred = _tc + _N * _P
        lnp    += -0.5 * np.sum((_T_obs - _T_pred) ** 2 / _sigma ** 2)
```
